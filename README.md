# RhemaNotes
Instant Sermon Summaries & Study Tools

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/2f0b5608-bfa2-4385-b504-b31c84f7e5c9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
